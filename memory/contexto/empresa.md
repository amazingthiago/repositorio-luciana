# Contexto Institucional — Amazing PRO

Referência rápida sobre a empresa, para o Claude entender o pano de fundo institucional em qualquer conversa deste repositório.

## O que é a Amazing PRO

Empresa brasileira de **MaaS (Methodology as a Service)**: desenvolvimento de liderança, cultura organizacional e educação executiva. Não é consultoria tradicional, não é EdTech, não é SaaS puro — é um modelo de metodologia proprietária entregue como serviço/produto.

## Onde vive o quê

| Repositório | Conteúdo |
|---|---|
| `amazingthiago/amazing-skills` | Skills, agentes e metodologias oficiais — fonte da verdade de *como* a Amazing PRO trabalha |
| `amazingthiago/repositorio-*` | Repositórios pessoais de cada membro do time (memória de trabalho + práticas individuais) |

## Como este repositório se conecta ao resto

Este é o repositório pessoal de Luciana. Ele **não substitui** o `amazing-skills` — quando uma tarefa exigir metodologia oficial (ROMA, Destilador, diagnóstico cultural, etc.), a fonte é o repositório mestre. Este repositório aqui guarda o contexto de *quem* Luciana é, com quem trabalha e como prefere operar.
