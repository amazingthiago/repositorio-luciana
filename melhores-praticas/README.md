# Melhores Práticas — Luciana

Cada arquivo aqui é um processo, aprendizado ou forma de trabalhar que Luciana quer que o Claude siga de forma **consistente**, em vez de reexplicar toda vez.

## Quando criar uma prática nova

- Você percebeu que está dando a mesma instrução pela terceira vez → vire prática.
- Você descobriu um jeito melhor de fazer algo (um formato de e-mail que funciona, uma forma de estruturar uma análise, um checklist antes de publicar algo) → vire prática.
- Um erro se repetiu e você quer que o Claude evite no futuro → vire prática.

**Regra prática:** se você se pegou pensando "eu já falei isso antes", é sinal de que devia estar aqui.

## Como criar

1. Copie `_template.md`.
2. Nomeie o arquivo em `kebab-case` descrevendo a prática (ex: `formato-relatorio-semanal.md`, `checklist-antes-de-publicar.md`).
3. Preencha as seções do template.
4. Se a prática for usada quase toda semana, adicione uma linha em `CLAUDE.md` na seção "Melhores práticas" apontando para ela.

## Convenções

- Um arquivo = uma prática. Não misturar processos diferentes no mesmo arquivo.
- Título do arquivo = nome da prática, sem ambiguidade.
- Prática que caiu em desuso: não apagar — mover a menção fora do CLAUDE.md, manter o arquivo aqui como histórico.
